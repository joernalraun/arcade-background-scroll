# set layer z index

Sets the z-index for one layer in the parallax scrolling stack.

```sig
scroller.setLayerZIndex(scroller._backgroundLayer(scroller.BackgroundLayer.Layer0), -1000)
```

The default z-indices for the parallax layers are:

* -1000 for layer 0 (bottom)
* -999 for layer 1
* -998 for layer 2
* -997 for layer 3
* -996 for layer 4 (top)

## Parameters

* **layer**: Chooses the parallax layer to set the image for
* **z**: The z-index for that layer to be drawn at
